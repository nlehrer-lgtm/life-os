Quarterly (or on-demand) deep review of goals. Scores progress, identifies stalled goals, and updates the plan forward.

## Steps

1. **Read current goals** from the goals file in `goals/`. Note the last updated date and current quarter.

2. **Read recent digests** from `digests/` since the last goals review (or last updated date). Scan for activity related to each goal.

3. **Score each goal** based on evidence from digests and activity:
   - **On track** — active progress, hitting milestones
   - **At risk** — some progress but behind schedule
   - **Behind** — little or no progress
   - **Abandoned** — no activity and no longer relevant
   - **Achieved** — goal met (celebrate it)

4. **Present the scorecard:**

```
## Goal Scorecard

### [Pillar Name]
- [Goal]: ON TRACK / AT RISK / BEHIND / ABANDONED / ACHIEVED
  Evidence: [What you found in digests and activity]

[etc.]
```

5. **For each "at risk" or "behind" goal**, ask a reflective question:
   - "This goal hasn't gotten much attention. Is it still important to you, or should we reprioritize?"
   - "What's blocking progress here? Is it time, motivation, or clarity?"
   - "Do we need to adjust the goal itself, or just the timeline?"

6. **Based on responses**, update the goals:
   - Revise quarterly milestones if needed
   - Update monthly targets for the upcoming month
   - Set this week's actions
   - Mark abandoned goals clearly (don't delete, strike through)
   - Add any new goals that emerged

7. **Save the updated goals file** with the new "Last updated" date and next review date.

8. **Summary** — end with a brief coaching note: what's going well, what needs attention, and one thing to focus on this week.

## Important

- **Evidence-based scoring.** Don't guess. If there's no evidence of progress in digests, it's "behind."
- **No judgment.** Behind doesn't mean failure. Life shifts priorities. The review is about realignment, not guilt.
- **Be direct but supportive.** Coaching, not cheerleading.
- **Suggest dropping goals** if they're clearly not happening and not important. Fewer focused goals beats many neglected ones.
- **Update, don't rewrite.** Preserve the structure and history. Achieved or abandoned goals stay in the file (marked accordingly) as a record.
