# Phonetic Pronunciation Guide

Best-guess phonetics for every name across all 131 cards / ~185 individuals. Nathaniel reviews → sends back corrections → I regenerate song lyrics using the corrected phonetic spellings (e.g. "Heidei" → spelled "Heidi" in the lyric block for Lyria 3 to pronounce correctly, while the canonical name stays "Heidei").

## Notation key

- **ALL CAPS** syllable = stressed
- "uh" = schwa (sounds like the "a" in *about*)
- "ee" = long e (*bee*)
- "ay" = long a (*day*)
- "eye" or final "-y" = long i (*pie*)
- "ow" = like *now*
- "oh" = long o (*toe*)
- "oo" = like *boot*

## How to correct

Reply with any of these formats — I'll batch the changes:

- `Bevere = buh-VAIR (not buh-VEER)`
- `Heidei = HY-dee` (spelling in the lyric will become "Heidi")
- `Marocco = muh-ROH-coh, accent first syllable`
