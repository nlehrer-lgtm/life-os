# Event Content — CONTEXT

**What it is:** A recurring [[Video Team]] responsibility at [[Messenger International]]. Pre-recorded and edited content tied to live events:
- **Promo** — content to advertise the event
- **In-event** — content shown at the event
- **Recap** — content that summarizes and recaps the event

Involves a lot of event footage and editing projects. Distinct from [[Live Events]] (real-time streaming).

**Current phase:** Folder established. SOP/templates TBD.
