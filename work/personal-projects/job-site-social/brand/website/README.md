# Jobsite Social — Website UI Kit

Click-through recreation of the public marketing site for Jobsite Social. The kit demonstrates the brand's visual system in context: dark hero, gold-accented CTAs, the 3-step process, pricing tiers, testimonials, and a dark footer.

## Run
Open `index.html` — single-file React + Babel prototype. All components are exported on `window` from `components.jsx`.

## Components

- `<Nav />` — black bar, gold CTA
- `<Hero />` — full-bleed dark hero with Bebas Neue headline and demo video placeholder
- `<StatBar />` — four gold stat counters on black
- `<HowItWorks />` — 3-step process (Show Up → Create → Post) with numbered cards
- `<Pricing />` — Standard vs Premium cards with "Recommended" gold badge
- `<Testimonial />` — gold quotation mark + client name + city
- `<Contact />` — dark CTA section with subscribe form
- `<Footer />` — black footer with link columns + social

## Interactive
The Subscribe CTA opens a small modal (fake) that you can dismiss. Plan selector toggles between Standard and Premium.
