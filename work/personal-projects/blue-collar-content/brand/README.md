## Quick "which file do I use?" guide

| You're putting the logo on… | Use this file |
|---|---|
| Website header, email signature, business card front | `horizontal-color` |
| IG / FB profile picture, favicon, app icon | `icon-color` |
| Square ad with limited width | `stacked-color` |
| Dark photo overlay or black background | `horizontal-white` |
| Embroidered hat, single-color shirt, etched metal | `icon-black` or `horizontal-black` |
| Stamp on the corner of a photo or video | `wordmark-color` |

## Fonts

- **Display:** [Anton](https://fonts.google.com/specimen/Anton) (free, Google Fonts)
- **Body:** [Barlow Condensed](https://fonts.google.com/specimen/Barlow+Condensed) (free, Google Fonts)

Install both before opening the SVGs in design software so the wordmarks render correctly.

## Before you send these to a printer or apparel vendor

The SVGs ship with **live text** so they're easy to edit. Most printers and embroiderers need text **converted to outlines/paths** before they'll run a job. Open the file in Illustrator, Affinity Designer, Inkscape, or Figma, select the text, and "Create Outlines" (Illustrator) / "Convert to Curves" (Affinity) / "Object → Path → Object to Path" (Inkscape). Save the outlined version with a `-outlined.svg` suffix.

If you want, ask me to produce outlined master files and I'll generate them.

## Need a PNG?

Open any SVG in Chrome or Safari, then export/screenshot at the size you need — or open in Figma/Illustrator and export at 1x, 2x, 3x. SVG is scale-infinite, so you can export at any resolution without quality loss.
