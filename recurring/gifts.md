# Gifts — Recurring

Date-anchored gift events: Valentine's Day, Christmas, Mother's/Father's Day, etc.

Birthdays and anniversaries are NOT listed here — those are pulled from `people/*.md` files by `/sync-recurring`. Add `**Birthday:** YYYY-MM-DD` or `**Anniversary:** YYYY-MM-DD` to the relevant person's file and the system handles the rest.

Format and rules: see [[recurring/CONTEXT]].
