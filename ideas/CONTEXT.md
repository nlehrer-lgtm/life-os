# Ideas — Context

## What This Is

A holding ground for Nathaniel's creative, story, and purpose-driven ideas — the ones that have been rattling around for a long time and could branch into many things. This is **not** goals (`goals/2026.md`) and **not** work (`work/`). It's the place where dreams, story seeds, and "I've always wanted to make this" sparks can land, be stored, and later be refined and built into something real.

Think: book ideas, film ideas, song ideas, testimony, related content, and the dreams underneath all of it. Each idea is allowed to be messy and unfinished. The point is to capture it before it's lost, then sharpen it over time.

This is creative and purpose work — it leans into [[Faith]] and [[Joy & Rest]] more than productivity. These are explicitly **not** practical/household ideas (no "reorganize the garage"). Those belong in `notes/inbox.md` or `recurring/`.

## What Lives Here

One folder or file per idea. An idea entry can hold any mix of:

- **Transcripts** — Nathaniel talking through the idea out loud
- **Written docs** — things he's already drafted or scribbled
- **Reference material** — clippings, links, images, songs that feed the idea
- **Branches** — where a single idea could split (a book *and* a film *and* a song)

## How to Use It

- **Capture first, organize later.** Don't force structure on a raw idea. Drop it in.
- When an idea matures enough to *act* on, it can graduate into a real project under `work/personal-projects/` or become a goal in `goals/2026.md` — link back here so the origin is preserved.
- Each idea should get its own file or folder once it has more than a stray line.
